# 宇宙旅人 - 部署指南

## 方式一：Vercel CLI 部署（推荐）

### 1. 安装 Vercel CLI
打开 PowerShell，运行：
```powershell
npm install -g vercel
```

### 2. 登录 Vercel
```powershell
vercel login
```
按照提示使用 GitHub、GitLab 或邮箱登录。

### 3. 部署项目
```powershell
cd e:\TRAE\tonight
vercel
```

### 4. 按提示操作
- 选择项目名称：`cosmic-travelers`
- 选择目录：`./`
- 是否修改设置：按回车保持默认
- 预览部署：输入 `y`

### 5. 获得分享链接
部署成功后，Vercel 会提供一个 `.vercel.app` 结尾的 URL，可以直接分享。

---

## 方式二：GitHub + Vercel 网页部署

### 1. 创建 GitHub 仓库
- 访问 https://github.com
- 创建新仓库 `cosmic-travelers`

### 2. 推送代码
在项目目录运行：
```powershell
cd e:\TRAE\tonight
git init
git add .
git commit -m "宇宙旅人 V0.2"
git branch -M main
git remote add origin https://github.com/你的用户名/cosmic-travelers.git
git push -u origin main
```

### 3. 部署到 Vercel
- 访问 https://vercel.com
- 点击 "Add New Project"
- 选择刚创建的 GitHub 仓库
- 点击 "Deploy"
- 等待部署完成，获得分享链接
